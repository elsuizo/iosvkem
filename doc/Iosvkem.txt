		  Iosvkem color scheme    by neutaaaaan

Iosvkem, a color scheme with a unique touch.

==============================================================================
Color scheme options					Iosvkem-options*

						*g:Iosvkem_transp_bg*
Set to 1 if you want a transparent background. Takes effect only in the
terminal.
>
	let g:Iosvkem_transp_bg = 0
<
						*g:Iosvkem_disable_italic*
Set to 1 if you want to disable italicized characters. Potentially useful if
you can't get your environment to behave, but recommended against. Takes
effect only in the terminal.
>
	let g:Iosvkem_disable_italic = 1
<
